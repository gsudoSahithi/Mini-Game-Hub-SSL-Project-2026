# MiniGameHub — Usage Guide

## Installation and Running






1. Download the project `.tgz` file.
2. Extract it using:
```bash
   tar -xvzf submission.tgz
```
3. Open a terminal and navigate to the extracted folder:
```bash
   cd submission/hub
```
4. Run the application:
```bash
   bash main.sh
```

---
## User Login and Registration
When the application starts, you will be prompted to either register a new account or log in.
Two players are required to enter their usernames and passwords to proceed.

## Important Notes:

If a username already exists, you will be asked to choose a different one.
If the password confirmation does not match during registration, you will need to re-enter it.
## Game Selection Screen
After successful authentication, the main game menu will appear.
You can select any available game through the graphical interface.
## Playing the Game
The chosen game will launch in a Pygame window.
Follow the on-screen instructions to play.
## After Completing a Game
Once a match finishes, you will be redirected back to the main menu.
From there, you can:
Start another game
Exit the application





# Mini-Game-Hub-SSL-Project-2026
SSL Project Multi-user game hub that integrates Bash scripting for authentication and Python (Pygame) for gameplay. Two authenticated players select a game from a menu, play via a graphical interface, and have their results recorded on a persistent leaderboard.
